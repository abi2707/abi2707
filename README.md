**[SkinScan AI](https://github.com/abi2707/skinscan-ai) — Real-Time Skin Analysis**
Webcam-based skin type classifier using OpenCV. Analyses brightness, texture variance, redness, and tone uniformity to classify Oily/Dry/Combination/Sensitive/Normal. Outputs a full personalised skincare routine with product recommendations.

**[RAG + Episodic Memory for Dialogue](https://github.com/abi2707/rag-episodic-memory)**
Benchmarked 6 retrieval strategies (BM25, Dense, Hybrid, Cross-Encoder, Memory-Augmented) on 74k utterances from DailyDialog + PersonaChat. Memory-augmented approach uses Ebbinghaus forgetting curves to filter low-retention context before re-ranking. Best MRR: 0.3116.

Adaptive anomaly detection for agricultural sensor streams. Compares static vs. drift-aware models (ADWIN, Page-Hinkley) across 7 simulated seasons. Includes a Streamlit dashboard and auto-generated PDF report.
